# Keywords and Phrases for ClassifierEngine

**Purpose:**  
This document contains the initial set of keywords and short phrases used by the ClassifierEngine to determine Red and Yellow issues. It is designed specifically for USA residential home building projects.

**Phase 1 – Foundation**  
This is the first version of the keyword library. It was built using common language from USA home builders and field subcontractors.

**Future Improvements**  
As the system is used, we will expand and refine this list based on:

- Real messages coming through SMS, Email, Voice, and MMS from actual subcontractors.
- Feedback and corrections from the Project Manager.
- Patterns observed across multiple builders once more companies start using the system.

This document is intended to be a **living file** that improves over time with real-world usage.
